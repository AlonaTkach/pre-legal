# PL-7 — Support multiple users and final polish

**Type:** Feature
**Status:** Done

## Summary

Add a proper sign in and sign up screen that lets the user register and come
back into the platform. Store previously generated documents and allow the user
to look back at prior documents they've created.

## Details

- The database can be temporary and be reset every time the server restarts.
- The platform is usable without logging in (freemium pattern); signing in lets
  the user **save** and revisit documents.
- Add **polish** to all screens so they look like a professional SaaS
  application (use the brand color scheme in CLAUDE.md).
- Add a **disclaimer** that the documents should be considered drafts and are
  subject to legal review.
