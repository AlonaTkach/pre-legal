# pre-legal

A platform for drafting common legal agreements.

> pre-legal does **not** replace a lawyer. It does the pre-work so that an
> attorney or paralegal can be best set up — or serves as a tool a lawyer uses.

## Status

Project in progress — will be completed in one week.

## Structure

- `templates/` — legal agreement templates (Markdown), sourced from
  [Common Paper](https://github.com/CommonPaper) under a Creative Commons license.
- `catalog.json` — machine-readable index of the available templates.
- `front-end/` — Next.js application for drafting agreements (e.g. mutual NDA).
- `tasks/` — local task descriptions (used in place of a Jira board).
